# Accountant User Stories
<br>
<br>


## Payments and Receipts
- As an Accountant, I want to record customer payments so that the daily cash balance is accurate.
- As an Accountant, I want to issue receipts for payments so that transactions are properly documented.
- As an Accountant, I want to track payments and proceeds by date and customer so that financial reports are accurate.
- As an Accountant, I want to look up orders by their ID so that I can correctly initialize prepayments or verify payment status.
<br>
<br>

## Bank Management
- As an Accountant, I want to record bank deposits and withdrawals so that the system reflects actual account balances.
- As an Accountant, I want to reconcile bank statements with system records so that discrepancies are identified and resolved.
<br>
<br>

## Credits
- As an Accountant, I want to manage customer credits so that outstanding balances are properly tracked.
- As an Accountant, I want to apply payments to credits automatically so that customer accounts remain consistent.
<br>
<br>

## Daily Cash
- As an Accountant, I want to verify the daily cash balance so that the store’s financial records are up-to-date.
- As an Accountant, I want to generate daily cash reports so that management can review store performance.
<br>
<br>

## Reporting
- As an Accountant, I want to export financial data to Excel or PDF so that I can create reports for management or auditors.
- As an Accountant, I want to track all financial transactions by type (payment, receipt, credit) so that reporting is comprehensive.
<br>
<br>

## Marketplace Order Reconciliation
- As an Accountant, I want to receive PDF files from marketplaces (e.g., Skroutz) containing order payments so that I can reconcile them with system records.
- As an Accountant, I want to look up each order by its ID and update the client’s balance:  
  - Null the balance if the order is paid.  
  - Mark the order as pending if payment is not received.  
- As an Accountant, I want this reconciliation workflow to be as fast and efficient as possible so that financial records remain up-to-date without delays.
