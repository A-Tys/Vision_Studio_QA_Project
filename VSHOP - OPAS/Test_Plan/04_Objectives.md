## Test Objectives
<br>
<br>

**The main objectives of this testing effort are to**:
<br>
<br>

**1. Assess Functional Accuracy**

* Verify that each module behaves as expected.
* Identify features that are broken, inconsistent, or missing.
<br>
<br>

**2. Evaluate Usability**

* Review system navigation and user interactions.
* Determine areas where workflows could be simplified or improved.
<br>
<br>

**3. Identify Security Concerns**

* Check login functionality and access control.
* Perform basic input validation and simple script injection tests.
<br>
<br>

**4. Provide a Comprehensive System Overview**

* Document which functionalities are actively used and which are redundant or rarely used.
* Highlight gaps in the system and potential areas for improvement.
<br>
<br>

**5. Support Decision-Making for Improvements**

* Deliver actionable insights for modernization and system enhancement, if required.
