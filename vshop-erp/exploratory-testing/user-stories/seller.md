# Seller User Stories
<br>
<br>

## Sales Orders
- As a Seller, I want to create new sales orders so that customer purchases are processed quickly.  
- As a Seller, I want to edit or cancel sales orders before they are finalized so that mistakes can be corrected.  
- As a Seller, I want to generate sales documents (invoices, receipts) so that transactions are properly documented.  
<br>
<br>

## Offers & Customizations
- As a Seller, I want to create customized offers (e.g., hardware bundles with different configurations) so that clients can choose the best option.  
- As a Seller, I want to look up an offer by its ID so that I can quickly access details during negotiations.  
<br>
<br>

## Customer Management
- As a Seller, I want to search for a customer by name, phone number, or ID so that I can easily find their records.  
- As a Seller, I want to create a new **customer profile** in the system so that new clients are properly registered.  
- As a Seller, I want to look up orders by ID so that I can track the status and history of specific purchases.  
- As a Seller, I want to apply credits or discounts to a customerâ€™s order so that special agreements are respected.  
- As a Seller, I want to track outstanding payments from customers so that I can remind them of pending balances.  
<br>
<br>

## Communication
- As a Seller, I want to communicate with customers via the portal (messages, live chat) so that support is centralized.  
- As a Seller, I want to log customer requests or complaints so that issues can be escalated to the appropriate department.  
<br>
<br>

## Coordination with Warehouse
- As a Seller, I want to check stock availability before confirming an order so that I avoid overselling.  
- As a Seller, I want to notify the Warehouse Manager when an item is sold so that stock records are updated.  
<br>
<br>

## Daily Operations
- As a Seller, I want to review my daily sales summary so that I can track my performance.  
- As a Seller, I want to manage returns and exchanges with customers so that they are handled efficiently and according to policy.
