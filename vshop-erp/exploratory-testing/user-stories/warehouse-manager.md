# Warehouse Manager User Stories
<br>
<br>

## Inventory Management
- As a Warehouse Manager, I want to monitor current stock levels so that inventory is always accurate.
- As a Warehouse Manager, I want to update stock quantities after receiving shipments so that the system reflects real inventory.
- As a Warehouse Manager, I want to flag low-stock items so that the Buyer can reorder in time.
- As a Warehouse Manager, I want to mark items as sold and update real stock levels so that the inventory always reflects actual sales.
<br>
<br>

## Order Handling
- As a Warehouse Manager, I want to check purchase orders received from the Buyer so that incoming goods are prepared and stored correctly.
- As a Warehouse Manager, I want to process outgoing orders for customers or internal departments so that shipments are timely and accurate.
<br>
<br>

## Reporting
- As a Warehouse Manager, I want to generate stock reports so that management has visibility into inventory and warehouse performance.
