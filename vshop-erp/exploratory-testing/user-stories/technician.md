# Technician User Stories
<br>
<br>

## RMA (Return Merchandise Authorization)
- As a Technician, I want to accept defective products under warranty so that customers can get replacements or repairs.  
- As a Technician, I want to issue RMA numbers so that returned items are tracked properly.  
- As a Technician, I want to document the condition of a returned item so that warranty claims are supported with evidence.  
- As a Technician, I want to send defective items back to the producer so that warranty procedures are completed.  
- As a Technician, I want to track the status of RMAs (sent, under repair, returned) so that I can update customers.  
<br>
<br>

## Internal Service
- As a Technician, I want to diagnose issues with stock items before they are sold so that defective items are not delivered to customers.  
- As a Technician, I want to repair products returned by customers when possible so that costs and waiting times are minimized.  
- As a Technician, I want to record internal repairs in the system so that the product’s history is preserved.  
<br>
<br>

## External Service
- As a Technician, I want to send items to external service centers when in-house repair is not possible so that customers receive proper support.  
- As a Technician, I want to track external service jobs so that I know when to expect repaired items back.  
<br>
<br>

## Customer Support
- As a Technician, I want to log customer support requests so that issues are not lost.  
- As a Technician, I want to provide technical assistance (by phone, email, or in-shop) so that customers receive guidance.  
- As a Technician, I want to search for past customer support interactions so that I understand the history of an issue.  
<br>
<br>

## Coordination with Other Roles
- As a Technician, I want to notify the Warehouse Manager when a defective item is received so that stock records are updated.  
- As a Technician, I want to update the Seller when a customer’s product is repaired or replaced so that the customer can be informed.  
- As a Technician, I want to share defect trends with the Product Manager so that recurring issues are addressed with suppliers.  
<br>
<br>

## Reporting
- As a Technician, I want to generate reports on warranty claims and repairs so that management can analyze quality and costs.  
- As a Technician, I want to keep statistics on repair times and return rates so that efficiency can be measured.
