# Buyer User Stories
<br>
<br>

## Purchase Orders
- As a Buyer, I want to create purchase orders for suppliers so that stock is replenished in a timely manner.
- As a Buyer, I want to edit or cancel purchase orders before they are confirmed so that mistakes can be corrected.
- As a Buyer, I want to track the status of purchase orders so that I know which orders have been accepted, shipped, or delayed.
<br>
<br>

## Communication with Suppliers
- As a Buyer, I want to send messages to suppliers regarding orders so that any issues or clarifications can be resolved quickly.
- As a Buyer, I want to receive responses from suppliers in the system so that all communications are centralized.
<br>
<br>

## Warehouse & Inventory
- As a Buyer, I want to check stock levels in the warehouse before placing orders so that I avoid overstocking or stockouts.
- As a Buyer, I want to receive notifications about low stock so that I can place orders proactively.
<br>
<br>

## Service & Returns
- As a Buyer, I want to initiate RMA (returned merchandise) requests with suppliers so that faulty products can be replaced or repaired.
- As a Buyer, I want to track internal and external service requests related to purchased products so that issues are resolved efficiently.
<br>
<br>

## Supplier Comparison and Contact
- As a Buyer, I want to compare prices of the same product from multiple suppliers quickly so that I can select the best option without manually checking each stock.  
- As a Buyer, I want to access detailed contact information for each supplier so that I can reach their managers immediately in case of urgent issues.
