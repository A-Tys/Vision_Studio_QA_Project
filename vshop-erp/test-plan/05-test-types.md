## Test Types

<br>

**1. Functional Testing**

* Verify that each feature behaves as expected.
<br>
<br>

**2. Non-Functional Testing**

* Usability Testing: Evaluate clarity, ease of navigation, and intuitiveness.

* GUI Testing: Ensure visual elements, labels, and layout are correct.

* Security Testing (Basic): Login, authentication, input validation, simple script injection.

* Compatibility Testing: Light testing on supported desktop browsers and OS.

* Database testing: (CRUD, data integrity, constraints, security checks).
<br>
<br>

**3. System Testing**

* Testing the complete integrated system to verify compliance with requirements.
<br>
<br>

**4. User Acceptance Testing (UAT)**

* Validate that the system meets user expectations and operational needs.
<br>
<br>

**5. Exploratory Testing**

* Initial unscripted testing to understand system behavior and discover potential issues.
<br>
<br>

**6. Positive and Negative Testing**

* Check the system works as expected with valid inputs (positive) and handles invalid inputs gracefully (negative).
