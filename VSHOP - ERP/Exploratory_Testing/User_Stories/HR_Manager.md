# HR Manager User Stories
<br>
<br>

## User Management
- As an HR Manager, I want to create, edit, or deactivate user accounts so that employees have appropriate system access.  
- As an HR Manager, I want to assign roles and permissions to users so that they can only access relevant modules.  
- As an HR Manager, I want to track user login activity so that I can monitor system usage and security compliance.  
<br>
<br>

## Access and Security
- As an HR Manager, I want to reset passwords and manage authentication settings so that users can securely access the system.  
- As an HR Manager, I want to review failed login attempts and security alerts so that potential unauthorized access can be prevented.  
<br>
<br>

## Reporting
- As an HR Manager, I want to generate reports of user activity and role assignments so that management can review compliance and resource allocation.  
<br>
<br>

## Communication Tracking
- As an HR Manager, I want to keep notes of phone calls and in-shop communications between employees and customers so that customer interactions are properly documented.  
- As an HR Manager, I want to keep notes of internal communications so that workplace interactions are tracked and any conflicts or issues can be reviewed.
